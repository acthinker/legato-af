Quick Start

To run the framework and development environment, there are two quick start options:

  • bootable USB stick
  • development host install.

You'll need to Prep Dev Kit Board beforehand.

Bootable USB Stick
Dev Host Install

Both options also require you to Setup Target.

After you've installed everything, you can start working with Legato Apps.

If you need to rebuild or customize the target, see Advanced topics.
